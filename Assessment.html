Pergunta 3

O elemento audio permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento audio e quando devem ser utilizados?


R: O elemento source é usado para especificar recursos de mídia de elementos de tag picture, audio, e vídeo em HTML5. É um elemento vazio. Geralmente é utilizado para disponibilizar formatos diferentes aceitos pro browsers diferentes.

Atributos source:
- src: Endereço do arquivo de media.
- type: O tipo MIME do arquivo, opcionalmente com um parametro de codecs.
- Media : Definição do media query desejado.

Pergunta 4

O elemento video permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento video e quando devem ser utilizados?


R:
Atributos video:
- autoplay: Usado para fazer com que um vídeo se inicie assim que possível sem parar de carregar dados.
- buffered: Serve para determinar os intervalos do vídeo.
- controls: Permite o usuário controlar a reprodução de vídeo.
- loop: Após o fim do vídeo, o atributo irá fazer com que seja reproduzido novamente.
- muted: Indica se o vídeo será inicializado sem som.
- src: A URL do vídeo a ser incorporado.
- height: A altura da área de exibição do vídeo, em pixels de CSS.
- poster: Uma URL indicando qual será a imagem capa do vídeo.
- Width: A largura da área de exibição, em pixels de CSS.

Pergunta 5

No caso do canvas, HTML 5 nunca, em momento algum, fez qualquer animação. Isso porquê o HTML 5 é uma simples linguagem de marcação. Dessa forma, o que é necessário para criar animações com HTML5? O que são APIs do HTML5?


R: É possível criar animações com HTML5 combinando HTML, CSS e JavaScript, com os quais você pode construir shapes. Além disso, é possível controlar animações e editar imagens, vídeo, e áudio por meio de elementos JS ou CSS, todos os quais você adiciona a uma "prancheta" de desenho, que você configura com o elemento "canvas". APIs são uma forma de criar aplicativos usando componentes pré-construídos. As APIs são muito significativas para o HTML5 e existe muitas dela para explorar, como "drag and drop", "Web Storage", "microdata" e "geolocation", entre outros.

Pergunta 6

Como o CSS3 pode ser aplicado na aparência da multimídia HTML5. Demonstre com um exemplo.

R: O CSS3 pode ser aplicado de algumas formas, é possível usá-lo para estilizar o player de vídeo, e torná-lo responsível em diferentes midias usando o media query, por exemplo.
